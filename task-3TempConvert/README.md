TempSphere — Smart Temperature Converter with Weather Spotlight

Overview  
TempSphere is an interactive web application built using HTML, CSS, and JavaScript. It converts temperature values between Celsius, Fahrenheit, and Kelvin while also featuring a simulated Weather Spotlight that displays random global cities with temperature updates.

The project uses a glassmorphism-based UI with gradients and responsive design for a modern user experience.

Purpose  
The project is designed to:
- Perform temperature conversions across standard units
- Practice JavaScript DOM manipulation
- Build interactive frontend functionality
- Simulate dynamic weather-style updates
- Improve UI/UX design skills using modern CSS

Technologies Used  
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

Core Features  

1. Temperature Conversion System  
Users can input a value and convert between:
- Celsius
- Fahrenheit
- Kelvin

Supported conversions include all standard unit combinations with instant results.

2. Weather Spotlight  
A dynamic panel that simulates live weather data by:
- Randomly selecting global cities
- Displaying country flags
- Generating temperature values
- Updating after each interaction

Sample cities include Tokyo, Paris, New York, Dubai, and London.

3. User Interface Design  
The UI is built using modern styling techniques:
- Glassmorphism effects
- Gradient backgrounds
- Blur and transparency layers
- Smooth hover animations
- Card-based layout

4. Responsive Design  
The application is fully responsive and optimized for:
- Desktop
- Tablets
- Mobile devices

JavaScript Functionality  

DOM Manipulation  
The application dynamically updates:
- Weather data
- City names
- Flags
- Conversion results
- Status messages

Conversion Logic  
Temperature conversion is handled using standard mathematical formulas with input validation to ensure correct results.

Input Validation  
The system checks for:
- Empty inputs
- Invalid numeric values

Invalid input triggers an error message instead of a result.
