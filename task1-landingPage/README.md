HireUp – Job Portal Landing Page

📌 Overview  
HireUp is a responsive job portal landing page built using HTML and CSS. It is designed to simulate a modern recruitment platform interface with multiple UI sections and interactive styling.

🚀 Features  

- Responsive navigation bar with logo and menu  
- Hero section with heading, description, and call-to-action buttons  
- Popular job categories displayed using cards  
- Featured jobs section with job listings and apply buttons  
- Dashboard preview layout for user activity tracking concept  
- "Why Choose HireUp" section highlighting platform benefits  
- Testimonials section for user feedback and trust building  
- Strong call-to-action section  
- Professional footer section  

🎨 UI & Styling  

- Flexbox and Grid layout system  
- Smooth hover effects and transitions  
- Gradient-based modern color palette  
- Card-based design structure  
- Clean and minimal user interface  

📱 Responsive Design  

The website is fully responsive and adapts to:  
- Desktop screens  
- Tablets  
- Mobile devices  

Media queries are used to ensure layout consistency across all screen sizes.  

🛠️ Tech Stack  

- HTML5  
- CSS3  

🎯 Purpose  

This project was built for learning and practicing front-end development skills including layout design, responsiveness, and UI structuring.  


📄 Sections Included  

- Header / Navigation Bar  
- Hero Section  
- Popular Categories  
- Featured Jobs  
- Dashboard Preview  
- Why Choose HireUp  
- Testimonials  
- Call-to-Action  
- Footer  



📌 Note  

This is a static frontend project and does not include backend functionality or real job application processing.
